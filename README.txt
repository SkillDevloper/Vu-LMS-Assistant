## Introduction

The **Vu LMS Assistant** is a Chrome extension designed to enhance the experience of students using the Virtual University Learning Management System (VU LMS). This tool simplifies access to various features of the LMS, helping users efficiently manage their educational resources.

## Features

- **Student Information Retrieval**: Automatically retrieves and displays student name, ID ...
- **Image Display**: Fetches and displays the user’s profile image from the LMS.
- **Page-Specific Functionality**: Tailors functionalities based on the current page within the VU LMS.

## Installation

To install the Vu LMS Assistant extension:

1. **Download the Extension**: Clone or download the repository containing the extension files.

2. **Open Chrome Extensions Page**:
   - Open Google Chrome.
   - Navigate to `chrome://extensions/`.

3. **Enable Developer Mode**:
   - Toggle the **Developer mode** switch in the upper right corner of the page.

4. **Load Unpacked Extension**:
   - Click on **Load unpacked**.
   - Select the folder where the extension files are located.

5. **Complete Setup**:
   - The extension will now be added to your Chrome browser. You can see its icon in the toolbar.

## Usage

1. **Access the Extension**:
   - Click on the **Vu LMS Assistant** icon in the Chrome toolbar.
  
2. **Subscription Login**:
   - Upon clicking, if you are logged in with your registered student ID (e.g., `bc220205492`), the extension will display relevant information. If you are not logged in, a notification will prompt you to subscribe.

3. **Page-Specific Operations**:
   - When on specific pages of the VU LMS, the extension will show or hide fields and buttons based on the context (e.g., Home, Lesson Viewer, GDB, Quiz).
